# University
Training project that uses:
+ Maven
+ Three-tier architecture
+ Spring Framework:
  + JDBC (It is planned to switch to JPA & Hibernate in future)
  + MVC
+ Unit testing:
  + JUnit5
  + Mockito
  + In-memory data base - H2DB
+ Frontend:
  + Thymeleaf
  + Bootstrap
+ Logging:
  + SLF4J + Logback
